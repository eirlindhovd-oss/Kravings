# Lunsjskeliste

En enkel og moro app for dele og stemme på lunsjforslag!

## Features
- Foreslå matvarer
- Stemme på favoritter
- Topp 3 forslag vises med medaljer

## Teknologi
- Frontend: HTML + CSS + JavaScript
- Database: Supabase (PostgreSQL)
- Hosting: GitHub Pages

## Bruk
1. Åpne [lunsjskelisten](https://eirlindhovd-oss.github.io/Kravings/)
2. Foreslå en matvare
3. Stem på favoritter
4. Topp 3 forslag vises med medaljer
